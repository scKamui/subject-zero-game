

# Subject Zero – Zombie Survival Game

## Overview
Subject Zero is a browser-based zombie survival game built using Phaser.  
The goal is to survive, defeat zombies, and reach the exit.

---

## How to Run the Game

### Step 1: Download the Project
Download and extract the project folder (`subject-zero-game`).

---

### Step 2: Open Terminal / Command Prompt

**Mac:**
- Press Command + Space  
- Type Terminal  
- Press Enter  

**Windows:**
- Press Windows Key + R  
- Type cmd  
- Press Enter  

---

### Step 3: Navigate to the Project Folder

Example:

cd subject-zero-game

(Or navigate to wherever you extracted the folder)

---

### Step 4: Start a Local Server

Make sure Python is installed, then run:

python3 -m http.server

**On Windows (if python3 doesn’t work):**

python -m http.server

---

### Step 5: Open the Game

Open a web browser (Chrome, Firefox, Edge, or Safari) and go to:

http://localhost:8000

The game should load automatically.

---

## Controls

- W / A / S / D → Move  
- Mouse → Aim  
- Left Click → Shoot  
- Enter → Start / Continue  
- P → Pause  

---

## Gameplay Tips

- Reach the exit to complete the level  
- Avoid zombies and manage positioning carefully  
- Tank zombies can kill you in one hit  
- Pick up weapons and damage boosts to survive longer  

---

## Notes

- The game must be run using a local server  
- Opening index.html directly may not work due to browser security restrictions  
- Tested on Chrome and Safari
